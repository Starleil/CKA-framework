# CKA-framework
This repository is an official implementation of the paper "Joint lesion detection and classification of breast ultrasound video via a clinicalknowledge-aware framework" (both the full code and dataset will be made publicly available upon acceptance of the manuscript).  

Environments
```bash
  python==3.8
  torch==1.11.0
```
# Dataset
This database is available for only non-commercial use in research or educational purpose. As long as you use the database for these purposes, you can edit or process images and annotations in this database.
